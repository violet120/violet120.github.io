補充：v-model

v-model 是畫面可以影響資料，而資料的改變也會影響畫面，也就是雙向控制 HTML 屬性，常見用於 input 上。

可以打開瀏覽器的開發者工具觀察(需要先安裝 vue.js devtools)，當使用者在 input 中輸入內容時，newInput 會同時出現相同的內容。
反之，當 newInput 有新內容時，input也會同時出現。
