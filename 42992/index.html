【學習筆記】hexo 設定

發表於 2023-08-10 分類於 Hexo

前言

在建構部落格中，Hexo X Next 主題的相關文章很多，只是設置完突然要更新，有些操作方式會忘記，寫一篇記錄方便未來翻閱。

基本設置

Site

在根目錄下的 _config.yml，# Site 區塊是設定 blog 的基本信息。

language 語言
預設是 en，台灣是 zh-TW。

timezone 時區
預設是自己電腦時間，基本上可以不用設定。
如果有需要修改，可參考 時區列表 選擇自己所在的時區，台灣是 Asia/Taipei。

更換 Hexo 主題

可以到 Hexo 官網提供的 themes，選擇感興趣的主題，我使用的是 Next 主題。
將主題下載下來後，放置在 themes 資料夾，到根目錄下的 _config.yml 更換要用的主題。

自訂樣式

需要先到 themes/hexo-theme-next/_config.yml 的 custom_file_path ，將 style 的註解拿掉。

按照 style 的路徑，新建資料夾跟檔案，在 styles.styl 撰寫 css 自行改變背景、按鈕樣式等。

✨ 自訂 Code 樣式

可以到 available themes 選擇想要的 Code 顯示顏色。

自訂 Favicon

默認圖示會是 Next 圖標，如果想要修改，可以到 themes/hexo-theme-next/source/images 中替換掉 favicon，放置新的圖標後，再到 主題目錄下 _config.yml 中找到 favicon，將路徑修改為剛剛新增的圖標即可完成替換。

使用 Favicon Generator 自訂圖標。

如果是想自製 Logo，可以使用 Designevo，將做好的 logo url 貼到 custom_logo。
